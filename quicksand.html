About the Font

This font was made by the creator Andrew Paglinawan in 2008. The font was made to be more of a display font but because of how it is so versatile and is very easy to read this font can be as a display or a non-display fonts.

The font Quicksand is a geometric font with rounded edges to make this a modern design for the font. To learn more about this font check out our social media pages today!!
